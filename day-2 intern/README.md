# To-Do List Web App

This is a simple To-Do List application built using HTML, CSS, and Vanilla JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Instant UI updates (no reload)

## Technologies Used
- HTML
- CSS
- JavaScript (DOM & Event Handling)

## How to Run
1. Download or clone repository
2. Open index.html in browser

## Learning Outcome
- DOM manipulation
- Event listeners
- Dynamic UI updates
